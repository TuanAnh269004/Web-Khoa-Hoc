Tổng quan khóa học

Khóa học này được thiết kế để đưa bạn từ con số 0 trở thành một nhà phát triển web full-stack có khả năng xây dựng các ứng dụng web phức tạp. Bạn sẽ học các công nghệ frontend và backend quan trọng nhất hiện nay.

Chúng tôi tập trung vào việc học thông qua thực hành, với rất nhiều dự án thực tế và bài tập coding. Bạn sẽ xây dựng một danh mục các dự án để trình bày kỹ năng của mình cho nhà tuyển dụng tiềm năng.

Bạn sẽ học được gì

  • Xây dựng các trang web đáp ứng với HTML5 & CSS3.
  • Lập trình tương tác với JavaScript và React.js.
  • Phát triển API backend với Node.js & Express.
  • Quản lý dữ liệu với cơ sở dữ liệu MongoDB.
  • Triển khai ứng dụng web lên máy chủ.
  • Kỹ năng làm việc nhóm và giải quyết vấn đề.

Nội dung khóa học

  • Bài 1.1: Web là gì và cách hoạt động
  • Bài 1.2: Cấu trúc cơ bản của tài liệu HTML
  • Bài 1.3: Thẻ HTML quan trọng (heading, paragraph, links, images)
  • Bài 2.1: Cách thêm CSS vào HTML
  • Bài 2.2: CSS Selectors vàSpecificity
  • Bài 2.3: Flexbox và CSS Grid
  • Bài 2.4: Responsive Design
  • Bài 3.1: Cú pháp cơ bản và biến
  • Bài 3.2: Điều kiện và vòng lặp
  • Bài 3.3: DOM Manipulation
